Proyek 2 — HollowWanderer

Judul: HollowWanderer – Adventure Game Design & Gameplay Concept
Genre: Adventure
Engine: Unity

Deskripsi Proyek

HollowWanderer adalah game adventure yang berfokus pada eksplorasi dan atmosfer dunia sunyi. Pemain berperan sebagai pengelana yang menjelajahi area kosong untuk menemukan petunjuk dan membuka cerita melalui lingkungan.

Game ini menekankan eksplorasi, atmosfer, dan pengalaman pemain, bukan pertarungan cepat.

Fitur Utama
	•	Eksplorasi berbasis area
	•	Puzzle ringan untuk membuka area baru
	•	Environmental storytelling
	•	Progress berbasis perjalanan

Peran Saya
	•	Merancang konsep game dan gameplay loop
	•	Mendesain alur eksplorasi dan level awal
	•	Mengevaluasi pengalaman pemain melalui playtesting

Nilai Proyek
	•	Adventure game design
	•	Gameplay loop planning
	•	Player experience design
